# Drone Project 

## About
A student‑built drone made for Hack Club. Designed with CAD, PCB, and wiring to show how hardware and code come together.

## Features
- Flies with stable control  
- Lightweight and reproducible design

## Why
Built to learn, share, and inspire. I wanted to prove students can design and assemble a working drone with clear steps.

## Usage
1. Clone the repo  
2. Check CAD files for the frame  
3. Follow PCB or wiring diagram
